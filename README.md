## Problem Statement

Modern airport baggage systems depend heavily on cloud infrastructure.  
If the network or cloud service fails, baggage operations can stop completely.

This leads to:

- Flight delays  
- Lost or misplaced baggage  
- Operational disruption at airports  

Airports need baggage systems that **continue operating even when the cloud is unavailable**.

---

## Our Solution

We built an **Edge-Resilient Baggage Operations System**.

The system follows an **edge-first architecture**, where all operational data is processed locally at the airport edge node.

Key idea:

- All baggage operations are written **to the edge first**
- The system **continues operating even during cloud outages**
- Data automatically **syncs to the cloud when connectivity is restored**
